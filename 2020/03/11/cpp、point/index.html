Class、Point、References

码俩段代码先自己体会一下int&与指针（不想好好写的原因是自己半夜码的原 md 不见了）

void Incream(int* value) { (*value)++; } int main() { int a = 5; Incream(&a); std::cout << a; std::cin.get(); }

这一段是指针用法

void Incream(int& value) { value++; } int main() { int a = 5; Incream(a); std::cout << a; std::cin.get(); }

前面俩个都是通过内存把a的值给++成6的
